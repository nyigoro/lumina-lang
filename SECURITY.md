# Security Policy

## Supported Versions

| Version | Supported |
| --- | --- |
| 0.5.x | Yes |
| Older versions | No |

## Reporting a Vulnerability

Please do not report undisclosed vulnerabilities in a public issue.

Send reports to `nyigoro@gmail.com` with:

- a short description
- affected version
- reproduction steps or proof of concept
- impact assessment if known

You can also use GitHub Security Advisories if private reporting is enabled for the repository.

## Response Goals

- initial acknowledgement: within 7 days
- status update after triage: as soon as a fix plan is clear
- coordinated disclosure after a fix or mitigation is available

## Scope

This policy covers:

- the Lumina CLI and compiler
- the runtime and stdlib
- the language server and VS Code extension
- publish and release tooling in this repository

For non-sensitive hardening ideas, bug reports, or docs corrections, use the normal issue tracker instead.
