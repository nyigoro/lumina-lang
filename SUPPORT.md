## Getting Help

Use the issue tracker for:

- bug reports
- feature requests
- documentation fixes
- editor and tooling issues

When asking for help, include:

- Lumina version
- OS and runtime
- command used
- minimal reproduction
- expected vs actual behavior

## Security

For private vulnerability reports, use `SECURITY.md` instead of opening a public issue.
