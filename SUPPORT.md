# Support

## Where to Start

- usage and setup: `docs/GETTING_STARTED.md`
- capabilities and status: `docs/CAPABILITIES.md`
- stdlib and runtime: `docs/STDLIB.md` and `docs/RENDER.md`
- browser and web-native direction: `docs/WEB_NATIVE_ROADMAP.md`

## Getting Help

Use the issue tracker for:

- bug reports
- feature requests
- documentation fixes
- editor and tooling issues

When asking for help, include:

- Lumina version
- OS and runtime
- command used
- minimal reproduction
- expected vs actual behavior

## Security

For private vulnerability reports, use `SECURITY.md` instead of opening a public issue.
